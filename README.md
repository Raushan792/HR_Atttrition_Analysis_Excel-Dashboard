# 📊 HR Attrition Analysis Dashboard

An interactive Excel dashboard that analyzes employee attrition patterns across departments, gender, and satisfaction metrics — helping HR teams identify key drivers of employee turnover.

---

## 🖼️ Dashboard Preview

![Dashboard Overview](dashboard_overview.png)

---

## 🎯 Objective

To analyze employee attrition data and uncover actionable insights that help organizations reduce turnover, improve employee satisfaction, and make data-driven HR decisions.

---

## 🔍 Key Insights

- 📌 **Marketing** has the highest attrition rate at **11.94%**
- 📌 **Sales** department has the highest attrition count — **13 employees**
- 📌 **HR department** shows the lowest overtime rate at **2.46%**
- 📌 **Engineering** employees have the highest average tenure — **4.95 years**
- 📌 Employees with **lower work-life balance scores** show higher attrition trends

## 📊 Dashboard Features

| Feature | Details |
|--------|---------|
| **Average Performance & Tenure** | Line chart comparing ratings across all departments |
| **Satisfaction Metrics** | Bar chart — Work-Life Balance, Job Involvement, Environment Satisfaction |
| **Rate Metrics** | Attrition Rate%, Overtime Rate% by Department |
| **Attrition by Department** | Bar chart showing total attrition count per department |
| **Attrition by Gender** | Pie chart breakdown — Male, Female, Other |
| **Interactive Slicers** | Filter by Department, Hire Date (2015–2024), Gender |
